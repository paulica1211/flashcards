# フラッシュカードアプリ - ユーザーガイド

## 概要

Google Sheetsと連携し、簡単にコンテンツを管理できるAndroidフラッシュカードアプリケーションです。リッチテキスト装飾、音声録音、滑らかな3Dフリップアニメーションでフラッシュカードを学習できます。

## 主な機能

### 1. フラッシュカード表示

- **スワイプで反転**: 左右にスワイプしてカードを反転し、答えを表示
- **3Dフリップアニメーション**: 滑らかなカード回転エフェクト
- **ナビゲーションボタン**: デッキ内を前後に移動
- **カードジャンプ**: 任意のカード番号に素早く移動

### 2. リッチテキスト装飾

Google Sheetsの複数のテキストスタイルをサポート：

- **太字** テキスト
- *斜体* テキスト
- <u>下線</u> テキスト
- ~~取り消し線~~ テキスト
- **色付きテキスト**（Google Sheetsの色を保持）

### 3. 音声録音・再生

- **音声録音**: カードごとに最大60秒の音声メモを録音
- **再生**: 録音した音声を再生
- **録音削除**: 不要な録音を削除
- **自動保存**: 音声ファイルを`Music/FlashcardApp/{シート名}/`に自動保存（バックアップ・共有用）

### 4. 学習管理

- **重要度レベル**: カードごとに優先度レベル（0～3）を設定
- **複数デッキ**: 複数のGoogle Sheetsに対応
- **進捗記録**: 各デッキでの現在位置を記憶
- **進捗リセット**: いつでも最初のカードに戻る

### 5. Google Sheets連携

- **リアルタイム同期**: Google Sheetsの変更がアプリに反映
- **オフライン対応**: ローカルキャッシュで高速アクセス
- **バッチプリフェッチ**: 次のカードを事前読み込みでスムーズなナビゲーション

## 使い方

### 初期設定

1. **アプリを起動**
2. **設定をタップ**（⚙️ アイコン）
3. **API URLを入力**: Google Apps ScriptのウェブアプリURLを貼り付け
4. **シートを選択**: 学習するシートを選択

### フラッシュカードの学習

1. **問題を表示**: カードの表面を読む
2. **カードを反転**: 左右にスワイプまたはタップして答えを表示
3. **ナビゲーション**: 矢印ボタンまたはスワイプでカード間を移動
4. **音声録音**: マイクアイコンをタップしてメモを録音
5. **音声再生**: 再生ボタンをタップして録音を聴く

### 設定メニュー

設定画面でできること：
- Google Sheetsのソースを変更
- 特定のカード番号にジャンプ
- 進捗をカード1にリセット
- 異なるフラッシュカードデッキに切り替え

## Google Sheetsのフォーマット

Google Sheetsは以下のように構成してください：

| A列 | B列 | C列 |
|----------|----------|----------|
| 問題 | 答え | 重要度（0～3） |
| 2+2は？ | 4 | 1 |
| フランスの首都は？ | パリ | 2 |

- **A列**: カードの表面（問題）
- **B列**: カードの裏面（答え）
- **C列**: 重要度レベル（0=低、3=高）
- **1行目**: ヘッダー
- **2行目以降**: フラッシュカードデータ

### テキスト装飾の適用

Google Sheetsで：
1. 装飾したいテキストを選択
2. 装飾を適用（太字、斜体、色など）
3. アプリが自動的に装飾を表示

## 必要な権限

アプリに必要な権限：
- **インターネット**: Google Sheetsに接続
- **音声録音**: 音声メモを作成
- **ストレージ**: 音声ファイルをMusicフォルダに保存

## 技術詳細

- **プラットフォーム**: Android 8.0+（API 26+）
- **バックエンド**: Google Apps Script + Google Sheets
- **アーキテクチャ**: RetrofitによるREST API
- **ストレージ**: ローカルキャッシュ + 音声用外部ストレージ

## 効果的な学習のヒント

1. **リッチ装飾を使用**: 太字や色でキーワードを強調
2. **発音を録音**: 語学学習に音声を活用
3. **重要度レベルを設定**: 難しいカードに優先順位をつける
4. **定期的に同期**: Google Sheetsを更新して最新コンテンツを維持
5. **ジャンプ機能**: さらに練習が必要な特定のカードを復習
