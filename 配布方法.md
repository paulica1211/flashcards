# フラッシュカードアプリの配布方法

## APKファイルの場所

ビルドされたAPKファイル：
```
C:\src\projects\flashcard\android\app\build\outputs\apk\debug\app-debug.apk
```

## 配布方法

### 1. ファイルを直接共有
- **メール添付**
- **クラウドストレージ** (Google Drive / OneDrive / Dropbox)
- **メッセージアプリ** (LINE / WhatsApp / Telegram)
- **USBケーブル**で直接転送

### 2. GitHub Releases（推奨）
1. GitHub → Releases → Create new release
2. APKファイルをアセットとして添付
3. URLを共有

### 3. Google Driveで共有
1. APKをGoogle Driveにアップロード
2. 共有リンクを作成
3. リンクを送信

## 受信者側の設定

### インストール手順
1. APKファイルをダウンロード
2. ファイルをタップしてインストール開始
3. 「この提供元を許可する」をオン（Android 8.0以降）
4. インストールを続行

### 古いAndroidの場合（7.1以前）
1. 設定 → セキュリティ
2. 「提供元不明のアプリ」をオン
3. APKをインストール

## Google Sheetsの設定（必須）

**重要:** 各ユーザーは自分のGoogle Sheetsを設定する必要があります。

### 1. Google Sheetsの作成
- **列A**: カードの表面
- **列B**: カードの裏面
- **列C**: 重要度（0-3）
- **1行目**: ヘッダー
- **2行目以降**: データ

### 2. Google Apps Scriptの設定
1. Google Sheets → 拡張機能 → Apps Script
2. `google-apps-script/Code.gs`の内容をコピー＆ペースト
3. デプロイ → 新しいデプロイ
4. 種類を選択 → ウェブアプリ
5. 実行ユーザー: 自分
6. アクセスできるユーザー: 全員
7. デプロイ → URLをコピー

### 3. アプリの設定
1. アプリを開く
2. Settings（設定）ボタンをタップ
3. API URLを入力（Google Apps ScriptのURL）
4. シート名を選択

## 主な機能

### カード表示機能
- **スワイプでカードを反転**: 左右にスワイプでカードを裏返し
- **3Dフリップアニメーション**: 滑らかなカード反転アニメーション
- **前後カードナビゲーション**: ボタンで前後のカードに移動

### テキスト書式対応
- **太字（Bold）**: Google Sheetsの太字書式を表示
- **斜体（Italic）**: 斜体書式を表示
- **下線（Underline）**: 下線書式を表示
- **取り消し線（Strikethrough）**: 取り消し線書式を表示
- **色（Color）**: テキスト色を表示

### 音声録音・再生機能
- **音声録音**: カードごとに最大60秒の音声メモを録音
- **音声再生**: 録音した音声を再生
- **録音削除**: 不要な録音を削除
- **公開フォルダへの自動保存**: Music/FlashcardApp/{シート名}/フォルダに自動的に音声ファイルを保存（バックアップ・共有用）

### 学習管理機能
- **重要度レベル**: カードごとに0-3の重要度を設定可能
- **シート別進捗管理**: 複数のフラッシュカードセットに対応
- **カード番号指定ジャンプ**: 設定画面から任意のカード番号にジャンプ
- **進捗リセット**: カード1に戻す機能

### データ連携
- **Google Sheetsとの連携**: リアルタイムで同期
- **オフライン対応**: キャッシュとローカルストレージで高速動作
- **バッチプリフェッチ**: 次のカードを事前読み込みで快適な操作

## トラブルシューティング

### インストールできない
- 提供元不明のアプリを許可しているか確認
- ストレージ容量を確認

### カードが表示されない
- Google Sheets URLが正しく設定されているか確認
- インターネット接続を確認
- Google Apps Scriptが正しくデプロイされているか確認

### カード数が間違っている
- Google Sheetsの列Aに空白行がないか確認
