# フラッシュカードアプリの配布方法

## APKファイルの場所

ビルドされたAPKファイル：
```
C:\src\projects\flashcard\android\app\build\outputs\apk\debug\app-debug.apk
```

## 配布方法

### 1. ファイルを直接共有
- **メール添付**
- **クラウドストレージ** (Google Drive / OneDrive / Dropbox)
- **メッセージアプリ** (LINE / WhatsApp / Telegram)
- **USBケーブル**で直接転送

### 2. GitHub Releases（推奨）
1. GitHub → Releases → Create new release
2. APKファイルをアセットとして添付
3. URLを共有

### 3. Google Driveで共有
1. APKをGoogle Driveにアップロード
2. 共有リンクを作成
3. リンクを送信

## 受信者側の設定

### インストール手順
1. APKファイルをダウンロード
2. ファイルをタップしてインストール開始
3. 「この提供元を許可する」をオン（Android 8.0以降）
4. インストールを続行

### 古いAndroidの場合（7.1以前）
1. 設定 → セキュリティ
2. 「提供元不明のアプリ」をオン
3. APKをインストール

## Google Sheetsの設定（必須）

**重要:** 各ユーザーは自分のGoogle Sheetsを設定する必要があります。

### 1. Google Sheetsの作成
- **列A**: カードの表面
- **列B**: カードの裏面
- **列C**: 重要度（0-3）
- **1行目**: ヘッダー
- **2行目以降**: データ

### 2. Google Apps Scriptの設定
1. Google Sheets → 拡張機能 → Apps Script
2. `google-apps-script/Code.gs`の内容をコピー＆ペースト
3. デプロイ → 新しいデプロイ
4. 種類を選択 → ウェブアプリ
5. 実行ユーザー: 自分
6. アクセスできるユーザー: 全員
7. デプロイ → URLをコピー

### 3. アプリの設定
1. アプリを開く
2. Settings（設定）ボタンをタップ
3. API URLを入力（Google Apps ScriptのURL）
4. シート名を選択

## アプリのビルド方法

### Debug APK（開発用）
```bash
cd C:\src\projects\flashcard\android
gradlew.bat assembleDebug
```

### Release APK（配布用・推奨）
```bash
cd C:\src\projects\flashcard\android
gradlew.bat assembleRelease
```

## 主な機能

- **スワイプでカードを反転**: 左右にスワイプ
- **3Dフリップアニメーション**
- **重要度レベル**: 0-3で設定可能
- **シート別進捗管理**: 複数のフラッシュカードセットに対応
- **Google Sheetsとの連携**: リアルタイムで同期
- **オフライン対応**: キャッシュとローカルストレージ

## トラブルシューティング

### インストールできない
- 提供元不明のアプリを許可しているか確認
- ストレージ容量を確認

### カードが表示されない
- Google Sheets URLが正しく設定されているか確認
- インターネット接続を確認
- Google Apps Scriptが正しくデプロイされているか確認

### カード数が間違っている
- Google Sheetsの列Aに空白行がないか確認
